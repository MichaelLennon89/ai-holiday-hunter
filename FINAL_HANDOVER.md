# AI Holiday Hunter - launcher handover

This build fixes the startup gap from earlier versions.

## What is new
- one-click self-bootstrapping launcher
- desktop-app launcher
- forced refresh launcher
- browser logs and server logs
- automatic port selection if 8000 is busy
- Playwright Chromium install on first run

## Best way to run
### Windows browser mode
Double-click `START_AI_Holiday_Hunter.bat`

### Windows desktop mode
Double-click `START_AI_Holiday_Hunter_Desktop.bat`

### Force a clean refresh
Double-click `INSTALL_OR_REFRESH.bat`

## Notes
The app still runs locally. The desktop mode wraps the same local website in a native window.
