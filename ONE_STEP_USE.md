# One-step use

## Windows
Double-click `START_AI_Holiday_Hunter.bat`.

That single launcher now:
- creates `.venv` if missing
- installs or refreshes dependencies
- installs the Playwright Chromium browser
- starts the local app
- opens it in your browser

## Desktop-app style window
Double-click `START_AI_Holiday_Hunter_Desktop.bat`.

This tries to open the app inside a native desktop window. If the desktop window layer is not available, it falls back to your normal browser.

## Refresh everything
Double-click `INSTALL_OR_REFRESH.bat`.

Use this if you want to force a clean dependency refresh.
